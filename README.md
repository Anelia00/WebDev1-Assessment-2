# WebDev1-Assessment-2
This repository contains my work for **Assessment 2: Web Development Project**. The assessment focuses on designing and developing a multi-page website for a Bristol-based planetarium called BigPlanetarium.

The website includes three main pages: **a homepage, a Milky Way & planets page, and a Saturn page.** It applies key web development concepts such as **Semantic Markup, CSS Styling, Page Layout, Interactivity with JavaScript, and Web Accessibility.**

Each page presents content using a clear structure, simple navigation, and visual elements such as images and video. The aim of the assessment is to demonstrate an understanding of building a complete website, including clean code, responsive design, accessibility features, and overall user experience.
